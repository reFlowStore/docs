<br />
<Image src="/static/hunting_banner.png" alt="Hello" width={1248} height={0} style={{ borderRadius: '8px' }} />
<br />

Guide to install reFlow Hunting.


<Steps>
### Download Resource
 
To get started first download your asset previously purchased from [reflow-store.com](https://reflow-store.com/) login into [keymaster](keymaster.fivem.net) and you will find your resource under the option **Granted Assets**
 
### Installation
 
Open the downloaded resource zip file and drag the folder into your servers resources folder

### Starting Order

```lua
ensure reFlow-Bridge

ensure reFlow-Hunting
```
