# Installation

<Steps>
### Download Resource
 
To get started first download the bridge, it's included with any asset purchased from [reflow-store.com](https://reflow-store.com/) login into [keymaster](keymaster.fivem.net) and you will find the bridge under the option **Granted Assets**
 
### Installation
 
Open the downloaded resource zip file and drag the folder into your servers resources folder

### Starting Order

<Callout type="warning" emoji={<Warning className="pt-1" />}>
In order for the bridge to detect your resources and start correctly, you must start the resources before running the script
</Callout>

```lua
-- Any Supported Framework, Inventory, Target
ensure framework
ensure inventory
ensure target

-- Start Bridge After
ensure reFlow-Bridge

-- Start Script After Bridge
ensure reFlow-ScriptName
```
</Steps>
